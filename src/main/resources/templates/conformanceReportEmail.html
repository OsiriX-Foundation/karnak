How to read this report

Violation of the DICOM standard: missing/empty Type 1 attribute, missing mandatory module, wrong VR or VM.
Recommended-practice deviation: missing Type 2 attribute, unexpected enumerated value, consistency issue. An empty Type 2 attribute is conformant.
Informational only: retired attributes, unknown VR.

Notes Conditional modules and Type 1C/2C attributes are validated only when present (their conditions are free text in the standard). Pixel data and other bulk values are not inspected; private tags are not validated.
